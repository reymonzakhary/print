# How to correctly use the Repository Pattern

## Repository Pattern

The Repository Pattern is a design pattern that provides a way to manage data access and business logic in a structured manner. It separates the data access logic from the business logic of an application, allowing for more modular and maintainable code.

## How to correctly use the Repository Pattern

- Copy the -repositoryTemplate.js file
- Fill the index, create, read, update, and destroy functions, rename them if you want.
- Use the repository in the components
